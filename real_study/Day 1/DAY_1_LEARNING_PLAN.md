# ☀️ GO LEARNING - DAY 1 MASTER PLAN

**Thời gian:** 8 tiếng học (chia làm 4 session × 2 tiếng)
**Mục tiêu:** Nắm vững các khái niệm cơ bản của Go

---

## 📅 LỊCH TRÌNH CHI TIẾT

### ⏰ Session 1 (0-2 giờ): Nền tảng

- **Lesson 1:** Giới thiệu Go - Lịch sử, tính năng, cài đặt
- **Lesson 2:** Hello World - Setup dự án đầu tiên
- **Quiz:** 5 câu hỏi cơ bản

### ⏰ Session 2 (2-4 giờ): Biến & Kiểu dữ liệu

- **Lesson 3:** Biến, hằng, kiểu dữ liệu cơ bản
- **Lesson 4:** String, Array, Slice
- **Mini Project 1:** Tính toán tuổi + chỉ số BMI

### ⏰ Session 3 (4-6 giờ): Điều khiển & Hàm

- **Lesson 5:** Control Flow (if/for/switch)
- **Lesson 6:** Functions - Khai báo, tham số, return
- **Mini Project 2:** Game đoán số

### ⏰ Session 4 (6-8 giờ): Cấu trúc dữ liệu & Ôn tập

- **Lesson 7:** Maps & Structs
- **Lesson 8:** Ôn tập + Q&A
- **Mini Project 3:** Quản lý danh bạ (Contact Manager)
- **Final Quiz:** 10 câu hỏi

## 📋 MINI PROJECTS

### Project 1: BMI Calculator (1 giờ)

- Input: cân nặng (kg), chiều cao (m)
- Output: BMI + nhận xét (gầy/bình thường/béo)
- Kỹ năng: Variables, Functions, If/Else

### Project 2: Number Guessing Game (1.5 giờ)

- Máy tính pick random number (1-100)
- User đoán, máy feedback (too high/low/correct)
- Đếm số lần đoán
- Kỹ năng: Loop, Random, String Input

### Project 3: Contact Manager (2 giờ)

- CRUD operations: Add, View, Edit, Delete
- Lưu danh bạ trong Struct slice
- Menu lựa chọn
- Kỹ năng: Struct, Slice, Loop, Switch, Functions
