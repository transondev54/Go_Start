## 📋 MINI PROJECTS

### Project 1: Task Manager with File Persistence (1.5 giờ)

- CRUD operations: Add, View, Update, Delete tasks
- Lưu tasks vào file JSON
- Load tasks từ file khi khởi động
- Filter tasks by status (pending/completed)
- Kỹ năng: Pointers, JSON, File I/O, Error Handling

### Project 2: Weather App (2 giờ)

- Gọi OpenWeatherMap API
- Parse JSON response
- Display dữ liệu thời tiết (nhiệt độ, độ ẩm, vận tốc gió)
- Async calls sử dụng goroutines
- Lưu cache dữ liệu
- Kỹ năng: Goroutines, JSON, HTTP, Channels, Caching

### Project 3: Bank System with Database (2 giờ)

- Interfaces: Account (Savings, Checking, Investment)
- Database: SQLite với Go
- Operations: Deposit, Withdraw, Transfer, View History
- Transaction logging
- Concurrent transactions xử lý an toàn
- Kỹ năng: Interfaces, Database, SQL, Goroutines, Concurrency
